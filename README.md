# Coff Language

## Variables

### String variables

variableName = "value"

### Int variables

variableName = 10

### Expression variables

variableName = 10+20+30/40\*50

### Variable variables

variableName = otherVariableName

## Input

### Storing input in variables

< variableName

## Printing

### Printing strings

\> "Hello World"

Hello World

### Printing numbers

\> 42

42

### Printing expressions

\> 145+146-10\*20

91

### Printing variables

variableName = "value"

\> variableName

value

## Comments

### One-line comments

! This is a comment
